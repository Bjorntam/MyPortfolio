MY WORKS

PUNIC FAITH

Punic Faith is a text-based role-playing game that draws inspiration from the popular Dungeons and Dragons board game. It was created as the final project for a Computer Programming 3 course by a team of seven individuals (Ingemar Bjorn Raymundo, Lance Cedrick Solleza, Robert Gabriel Calla, Mark Laurenz Advincula, Juliana Rose Macaranas, Anna Marie Frio,and Sophia Marie Tagapan). In PunicFaith, players engage in gameplay driven by text, where they make decisions and interact with the game world through player input.

DEECOR Furniture Co

DEECOR Furniture Co. is an innovative web application developed using C#. This project, crafted as the culmination of my studies in Object-Oriented Programming (OOP), embodies an extensive e-commerce platform. This dynamic website boasts an array of functionalities, including seamless cart management, streamlined checkout processes, and secure payment gateways. Moreover, it features an intuitive admin interface, catering to the needs of website owners or administrators. Underpinning its robust functionality is a sophisticated database system leveraging Microsoft SQL technology, ensuring efficient data management and retrieval.
